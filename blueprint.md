
# Blueprint: MBTI 요괴 테스트 웹사이트 개선 프로젝트

## 1. 프로젝트 개요

본 프로젝트는 사용자의 MBTI 성향을 기반으로 자신만의 "요괴"를 찾아주는 독특한 컨셉의 웹 애플리케이션입니다. 사용자의 재미와 경험을 극대화하고, 동시에 사이트의 지속 가능한 성장을 위해 다음과 같은 기능 개선을 목표로 합니다.

## 2. 디자인 및 기능 명세

### 스타일 및 디자인
- **현대적이고 매력적인 UI:** 최신 웹 디자인 트렌드를 반영하여 사용자에게 시각적인 즐거움을 선사합니다.
- **인터랙티브 요소:** 부드러운 애니메이션과 전환 효과를 통해 사용자의 몰입감을 높입니다.
- **반응형 웹 디자인:** 모바일, 태블릿, 데스크탑 등 모든 기기에서 최적화된 화면을 제공합니다.

### 핵심 기능
- **MBTI 기반 요괴 매칭:** 16가지 MBTI 유형에 따라 각각 다른 요괴와 스토리를 제공합니다.
- **미니게임:** 사용자가 자신의 요괴와 상호작용할 수 있는 간단한 미니게임을 포함합니다.
- **요괴 진화 시스템:** 게임 내 재화(엽전)나 특정 조건을 만족하면 요괴가 진화하여 새로운 모습을 보여줍니다.
- **데이터 저장:** `localStorage`를 활용하여 사용자의 진행 상황(MBTI 결과, 요괴 상태, 재화 등)을 브라우저에 저장합니다.
- **SNS 공유:** 사용자가 자신의 요괴와 MBTI 결과를 카카오톡이나 인스타그램 스토리 등을 통해 친구들과 공유할 수 있습니다.
- **수익 모델 (광고):** 구글 애드센스를 연동하여 전면 광고와 보상형 광고를 통해 수익을 창출합니다.

## 3. 현재 요청사항 (v2)

- **구글 애드센스 최적화:**
    - **전면 광고:** 미니게임 종료 또는 요괴 진화 시점에 전면 광고 노출.
    - **보상형 광고:** "광고 시청 시 엽전 지급" 또는 "요괴 즉시 부활"과 같은 보상을 제공하여 사용자 자발적 광고 시청 유도.
- **검색 엔진 최_적화 (SEO):**
    - **콘텐츠 강화:** "MBTI 요괴 테스트", "나의 전생 요괴 찾기" 등의 키워드를 활용한 블로그형 콘텐츠 페이지 추가 제안. (장기적 관점에서 Next.js와 같은 프레임워크 전환 필요성 언급)
- **바이럴 마케팅 장치:**
    - **SNS 공유 버튼:** 결과 화면에 '카카오톡 공유하기', '인스타그램 스토리용 이미지 저장' 버튼 구현.
- **우선순위 개발 목록:**
    1.  **저장 기능 구현 (`localStorage`)**
    2.  **공유하기 버튼 추가**
    3.  **도메인 및 호스팅 (안내)**
    4.  **광고 연동 (준비 작업)**

## 4. 검색 엔진 최적화 (SEO) 기초 작업
- **목표:** 검색 결과에 더 잘 노출될 수 있도록 합니다.
- **작업 내용:**
    - HTML `<head>`에 사이트를 설명하는 `<meta name="description">`과 핵심 키워드를 담은 `<meta name="keywords">` 태그를 추가합니다.
    - `<title>` 태그를 "MBTI 요괴 테스트 - 나의 전생 요괴는?"과 같이 매력적으로 수정합니다.

## 5. 현재 요청사항 (v3) - 미니게임 전면 개편
- **목표:** 기존의 단발성 슬롯/뽑기 형태인 미니게임 3종을 상호작용이 있고 룰이 명확한 아케이드 형태로 개편.
- **활쏘기:** 
    - 3발을 주어 좌우로 이동하는 과녁을 직접 맞히는 물리적 슈팅 방식.
- **수수께끼:** 
    - 제한 시간 30초 내에 많은 문제를 푸는 타임어택 스피드 퀴즈로 전환 (정답 시 시간 연장, 오답 시 시간 차감).
- **윷놀이:** 
    - 일직선 10칸 트랙 레이스 게임 도입. 4번의 기회 안에 완주를 목표로 하며, 윷/모 시 추가 행동을 부여.
